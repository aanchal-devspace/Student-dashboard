# Student Study Dashboard

## About

Student Study Dashboard is a simple frontend web application made using HTML, CSS, and JavaScript. It helps students organize their study schedule and keep track of assignments.

## Features

* Add and delete subjects in the timetable
* Search subjects
* Add and manage assignments
* Mark assignments as completed
* Attendance calculator
* Current date and time
* Data saved using Local Storage

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage


## Project Structure

* `index.html` – Page structure
* `style.css` – Styling
* `logic.js` – JavaScript functionality
